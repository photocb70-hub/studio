
# Changelog

All notable changes to this project will be documented in this file.

## [2.50 alpha]
### Added
- AI Referral Letter Generator: Integrated a powerful Genkit flow to draft professional ophthalmology referral letters from clinical findings.
- IOP Correction Calculator: Added a precision tool to adjust Goldmann tonometry readings based on Central Corneal Thickness (CCT).

## [2.40 alpha]
### Added
- Ocular Pathology Guide Overhaul: Integrated a complete suite of clinical reference images for all major conditions, including AMD, Diabetic Retinopathy, Retinal Detachment, RVO, RAO, ERM, Anterior Uveitis, Corneal Ulcer, Scleritis, Optic Neuritis, Subconjunctival Haemorrhage, POAG, AACG, PCAG, Cataracts, Keratoconus, and Papilledema.
- Multi-Image Support: Updated the clinical lightbox to support multiple images per condition, enabling both internal (fundus) and external (front of eye) reference visualizations.
- Clinical Lightbox: Added a full-screen, high-resolution visualization tool for all pathology assets, providing an optimized experience for clinical detail recognition on mobile devices.
- Asset Precision: Synchronized the image registry with exact case-sensitive filenames.

## [2.10 alpha]
### Added
- AI Study Hub: Integrated PDF-to-Pro-Notes conversion using Genkit and `pdf-parse`.
- Study Vault: Firestore-backed storage for analyzed course units with persistent logging.

## [1.80 alpha]
### Added
- Rx Contact Lens Vertex Converter: Implemented a robust dual-meridian vertex distance compensation tool.
- Visual Acuity Converter: Added a clinical reference suite for converting between LogMAR, Snellen (m/ft), and Decimal standards.

## [1.50 alpha]
### Added
- AI Image Analyzer: Experimental tool for fundus photo analysis and feature highlighting.
- AI Problem Solver: Intelligent scenario analysis for complex optical non-tolerance cases.
- Ocular Drugs Guide: Reference guide for systemic drugs with ocular side effects.

## [1.00 alpha]
### Added
- Initial Release: Core dispensing calculators including Lens Thickness, Blank Size, Induced Prism, and Prescription Transposer.
- Modern UI: Responsive design with ShadCN components and "Optical" theme toggle.
