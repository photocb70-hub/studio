
# Changelog

All notable changes to this project will be documented in this file.

## [2.16 alpha]
### Added
- New Clinical Tool: Visual Acuity Converter. Supports Snellen (6m/20ft), LogMAR, and Decimal formats with a live reference table.
- Upgraded the Contact Lens Rx Converter to support full sphero-cylindrical vertex compensation (both meridians).

## [2.15 alpha]
### Added
- Established a new stability hardsave checkpoint.
- Verified cross-component consistency for the new `PowerAdjuster` tactile interface.

## [2.14 alpha]
### Fixed
- Fixed contrast issues in dark mode by adjusting `accent-foreground` HSL variables.
- Improved `PowerAdjuster` UI with larger, more distinct stepper buttons (+/-) for better mobile usability.
- Added explicit visual feedback for current power selection within the adjuster component.
- Standardized prescription results across all pages to ensure consistent high contrast in both themes.
